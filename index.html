Image Recognition Game

Machine learning and A.I are some things that I would love to do in the future. This application uses ML5's feature extractor. This class allows you to extract features of an image via a pre-trained model and re-train that model with new data. Using p5.js I created an HTML video that uses the video feed from the webcam. The user can capture and train three different images. The first image will be used to perform a jump in the game, the second image will be to shoot one of the incoming obstacles and the final image is to do nothing. Once the model is trained we can classify and finally play the game. This project was extremely enjoyable. Especially as it showed me the potential of machine learning. Specifically, image classification.
